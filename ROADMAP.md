# 🧬 CopyKit — Roadmap

**Vision:** The most alive, most beautiful frontend toolbox on the internet. Every other site shows a *picture* of a button — CopyKit lets you **play with the real thing, tweak it live, and copy it your way.**

## The 7 differentiators ("why didn't we think of that?!")
1. **Everything is alive** — real, interactive live previews (buttons hover, toggles flip, cursor effects follow your mouse).
2. **The Code Drawer** — click an item → a panel slides in with a big live playground + code in tabs (HTML · CSS · Tailwind · JS) → one-click copy.
3. **Tweak before you copy** — sliders adjust the item live (angle, radius, speed…) and the code updates with it.
4. **Build Your Kit** — star items into "My Kit" and export them all (saved in the browser, no login).
5. **Light & dark preview toggle.**
6. **⌘K command palette** — search everything instantly.
7. **The Aura** — custom cursor, premium motion, confetti on copy, a hidden easter egg. The unmistakable "Adham" stamp.

## Categories
- **Color:** Gradients · Palettes · Mesh · Animated backgrounds
- **Interactive:** Buttons · Toggles/switches · Custom cursors · Micro-interactions · Loaders · Hover effects · Cards (glass/neumorphism/clay)
- **Effects:** Shadows · Text effects · Glow tricks

## Phases
- **Phase 1 (flagship launch):** New engine — Code Drawer + live previews + tweak + multi-format copy + My Kit + ⌘K + aura. Launch categories: **Gradients, Buttons, Toggles, Cursor effects.**
- **Phase 2:** Shadows, Loaders, Text effects, Cards, Backgrounds.
- **Phase 3:** Deep links + per-item SEO pages, "surprise me," remix.
- **Phase 4:** Community submissions (people add their own → grows itself).

## Tech
Frontend-only, data-driven (each item is one entry). Static, fast, free, SEO-friendly. Migrate to a bigger setup only if it blows up.
